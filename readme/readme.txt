0. 준비
  가. mw_ahrs 기본 ss 설정은 0
  나. ros 패키지를 사용하려며 ss를 7로 변경
  다. 7은 가속도, 각속도, 각도 데이터
  라. 기본값
    1) MW-AHRS V1 or V2
      가) sp=100 : 송신주기 100ms
      나) sb=115200 : 115200bps
      다) st=1 : RS232 데이터전송타입(0-Binary, 1-Text)
      라) cmd=99 : 소프트웨어 리셋
      마) cmd=5 : euler 각도를 0으로 리셋
    2) MW-AHRS X1
      가) sp=500 : 송신주기 500ms
      나) sb=115200
      다) st=1
      라) cmd=99

1. 설정 순서
  가. MW-AHRS V1 or V2
    1) ss=0->7 : 데이터 전송 중단->가속도, 각속도, 각도 데이터 요청(시리얼)
    2) cmd=1 : 플래시 메모리 저장
  나. MW-AHRS X1
    1) ss=0->7 : 데이터 전송 중단->가속도, 각속도, 각도 데이터 요청(시리얼)
    2) sp=100 : 송신주기 100ms로 변경
    2) cmd=1 : 플래시 메모리 저장
